## 🚀 快速启动

### 方法一：使用启动脚本

**macOS/Linux:**
```bash
./start_web.sh
```

**Windows:**
```bash
start_web.bat
```

### 方法二：手动启动

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 启动服务
python3 app.py

# 3. 打开浏览器
访问 http://localhost:8887
```
