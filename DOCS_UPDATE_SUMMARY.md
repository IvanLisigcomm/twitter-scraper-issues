## 🎯 核心更新点

### 1. 端口号统一
- **旧值**：5000 或 8080
- **新值**：8887
- **原因**：避免macOS系统占用和常见冲突

### 2. 默认格式统一
- **旧值**：JSON
- **新值**：CSV
- **原因**：Excel友好，更适合数据分析

### 3. 预览功能增强
- **旧功能**：只能预览JSON
- **新功能**：支持CSV和JSON
- **优化**：优先显示CSV，自动格式识别
